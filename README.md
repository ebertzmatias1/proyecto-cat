# Proyecto C.A.T H.R.

Sitio web institucional de un club de liga regional.

## Descripcion del proyecto
Este sitio web cuenta con distintas secciones informativas, orientadas a socios, hinchas y público general.  
Se prioriza una experiencia de usuario simple, una estética acorde a los colores institucionales.

## Tecnologías
- HTML
- CSS
- Bootstrap 5
